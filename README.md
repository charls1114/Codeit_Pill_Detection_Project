# Codeit_Pill_Detection_Project
코드잇 초급 프로젝트  

.env 파일이 프로젝트 폴더에 있어야 합니다.

- .env 파일 내용 양식
```
# 데이터 경로 설정

# 구글 드라이브 폴더 경로
ORIGINAL_DATA_DIR=

# 프로젝트 폴더 경로
PROJECT_ROOT=
# 구글 드라이브에 있는 약 고유 id(original_id.json) 저장 파일 경로
ID_DATA_DIR=
# 구글 드라이브에 있는 YOLO 학습 설정 폴더(yolo_train_configs) 경로
CONFIG_DIR=
# SSD 학습 설정 폴더 경로
SSD_DIR= 
# 욜로 포맷 데이터셋 폴더(Pill_Detection_yolov8) 경로
YOLO_DATA_DIR=
# COCO 포맷 데이터셋 폴더 경로
COCO_DATA_DIR=
# 테스트 이미지 폴더 경로
TEST_DIR=
# YOLO 모델 학습 결과 저장 폴더 경로
RUN_DIR=
```
